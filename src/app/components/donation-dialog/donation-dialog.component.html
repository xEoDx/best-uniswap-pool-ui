Thanks for using Pool Analyzer

How does it work?

Check the Avg. gain / dollar provided column to know what pools are giving more comissions. The value is calculated based on the volume, tvl and the fee of the pool, but it doesn't take into account ranges. In order to do this calculation, every 12h we process all transactions done in the last 5 days on the Ethereum and Polygon blockchains.

The higher the avg. gain / dollar, the better. But please take into account that some pools are prone to have higher Impermanent Loss, so be aware of this and also consider having proper ranges on the pool, so that you keep earning comissions.
