Thanks for using PoolAnalyzer

How does it work?

Every 12h we compute all transactions done in the last 5 days on the Ethereum and Polygon networks, to be able to determine which is the best pool to provide liquidity. You can use this information to move your funds from existing pools or to create new positions based on the score parameter.

Avg. gain / dollar provided

This value is calculated based on the volume, tvl and the fee of the pool. This is based on the last 5 days, but it provides useful data on which are the pools that will give more commissions.
