Thanks for using Pool Analyzer

How does it work?

Pool Analyzer checks every 24h all the transactions done in Uniswap during the last 30 days in the Ethereum and Polygon networks.

Once the data is fetched, we calculate the average gain per dollar provided on each liquidity pool. Pool Analyzer uses the TVL, Volume and Fee of the pool to calculate the average gains per dollar provided on each liquidity pool.

The higher the avg. gain / dollar the more commissions you would earn, on average.

Be aware that some pools may be more attractive because of the possible commissions earned, but they may also be affected by the Impermanent Loss.

Also, this table does not take into consideration the ranges of the liquidity positions that you would provide. It is an average to be able to compare what pool would give more profits if we had the same range and the same quantity on both pools during the last X days.
