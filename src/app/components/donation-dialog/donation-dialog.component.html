How does it work?

Every 12h we compute all transactions done in the last 5 days Ethereum and Polygon networks to be able to determine which is the best pool to provide liquidity. You can use this information to move your funds from existing pools or to create new positions based on the score parameter.

Pool score

The score is calculated based on the volume, tvl and the fee tier of the pool. The higher the score, the more benefits you are going to get.

Server cost

Hosting this application has a cost. Any donation will help to mitigate the costs of maintaining the setup alive.
